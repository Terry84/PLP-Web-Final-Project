# Kilimanjaro Transporters Website

## Project Overview

Kilimanjaro Transporters is a professional transport services company based in Tanzania. This website is designed to advertise their business, showcase their services, share company history and mission, and provide an easy way for potential clients to contact them. The website is built using semantic HTML5, CSS for styling and responsiveness, and JavaScript for interactivity.

---

## Features

- Responsive layout optimized for mobile and desktop devices
- Consistent header and footer across all pages
- Navigation menu with toggle button for mobile devices
- Smooth scrolling for navigation links
- Interactive contact form with client-side validation and user feedback
- Animated UI elements on page load and user interaction
- Accessible design with ARIA attributes and keyboard focus management
- Clear typography and brand-aligned color palette
- Services section with descriptive icons and text

---

## Prerequisites

- A modern web browser (Chrome, Firefox, Edge, Safari) updated to latest versions
- Optional: A simple local HTTP server to serve files (recommended for best experience)
